# Keyboard Driver
USB keyboard driver for Linux.

## Install
```
git clone https://github.com/YehudaElyasaf/keyboard-driver/
cd keyboard-driver
make
sudo insmod keyboard_driver.ko
```

## Remove
```
sudo rmmod keyboard_driver
```
